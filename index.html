Core Chapters

Chapter 1: Descriptive Statistics

Data Analysis

Description: Understanding and summarizing data through measures of central tendency, dispersion, and shape.

Topics covered: Data types and variables, Measures of central tendency (mean, median, mode), Quartiles and percentiles, Measures of dispersion (variance, std dev), Skewness and kurtosis, Data visualization.

Python Implementation:
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Load dataset
data = pd.read_csv('dataset.csv')

# Descriptive statistics
mean_val = data['values'].mean()
median_val = data['values'].median()
std_val = data['values'].std()
skewness = data['values'].skew()

# Visualization
plt.figure(figsize=(12, 4))

plt.subplot(1, 3, 1)
plt.hist(data['values'], bins=30, edgecolor='black', alpha=0.7)
plt.title('Histogram')
plt.xlabel('Values')
plt.ylabel('Frequency')

plt.subplot(1, 3, 2)
sns.boxplot(y=data['values'])
plt.title('Box Plot')

plt.subplot(1, 3, 3)
data['values'].plot(kind='density')
plt.title('Density Plot')
plt.tight_layout()
plt.show()

print(f"Mean: {mean_val:.2f}, Median: {median_val:.2f}")
print(f"Std Dev: {std_val:.2f}, Skewness: {skewness:.2f}")

Chapter 2: Probability Theory

Theory

Description: Fundamental concepts of probability, rules, and theorems for quantifying uncertainty.

Topics covered: Definition of probability, Sample spaces, Addition/Multiplication rules, Conditional probability, Bayes' theorem, Independence.

R Implementation (Bayes' Theorem):
# Probability calculations in R
library(ggplot2)

# Bayes' Theorem Example
# P(Disease|Positive) = P(Positive|Disease) * P(Disease) / P(Positive)

sensitivity <- 0.95  # P(Positive|Disease)
specificity <- 0.90  # P(Negative|No Disease)
prevalence <- 0.01   # P(Disease)

# Calculate P(Positive)
p_positive <- sensitivity * prevalence + 
              (1 - specificity) * (1 - prevalence)

# Posterior probability
p_disease_given_positive <- (sensitivity * prevalence) / p_positive

cat("Probability of disease given positive test:", 
    round(p_disease_given_positive, 4), "\n")

Chapter 3: Random Variables & Distributions

Modeling

Description: Study of discrete and continuous random variables and their probability distributions.

Topics covered: Discrete/Continuous variables, PMF/PDF/CDF, Binomial, Poisson, Normal, Exponential, Uniform distributions.

Python Implementation (Distributions):
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import binom, poisson, norm

# Binomial Distribution
n, p = 20, 0.3  # n trials, p probability of success
x = np.arange(0, n+1)
pmf = binom.pmf(x, n, p)

plt.figure(figsize=(15, 4))

# Binomial Plot code...
plt.subplot(1, 3, 1)
plt.bar(x, pmf, alpha=0.7, color='steelblue')
plt.title(f'Binomial Distribution (n={n}, p={p})')

# Normal Plot code...
mu, sigma = 0, 1
x_normal = np.linspace(-4, 4, 100)
pdf_normal = norm.pdf(x_normal, mu, sigma)

plt.subplot(1, 3, 3)
plt.plot(x_normal, pdf_normal, linewidth=2, color='green')
plt.title(f'Normal Distribution (μ={mu}, σ={sigma})')
plt.tight_layout()
plt.show()

Chapter 4: Hypothesis Testing

Inference

Description: Statistical methods for making inferences and decisions about population parameters.

Topics covered: Null/Alternative hypotheses, Errors (Type I/II), P-values, Confidence intervals, Z-tests, T-tests, Chi-square, ANOVA.

Python Implementation:
import numpy as np
from scipy import stats

# One-Sample T-Test
# H0: population mean = 100
sample_data = np.array([98, 102, 95, 105, 99, 101, 97, 103, 100, 96])
population_mean = 100

t_statistic, p_value = stats.ttest_1samp(sample_data, population_mean)

print("One-Sample T-Test")
print(f"T-statistic: {t_statistic:.4f}")
print(f"P-value: {p_value:.4f}")

# Chi-Square Test
observed = np.array([[10, 20, 30], [6, 9, 17]])
chi2, p, dof, expected = stats.chi2_contingency(observed)

print("\nChi-Square Test")
print(f"Chi-square statistic: {chi2:.4f}")
print(f"P-value: {p:.4f}")

Chapter 5: Regression Analysis

Prediction

Description: Modeling relationships between variables for prediction and inference.

Topics covered: Simple/Multiple Linear Regression, Least squares, R-squared, Multicollinearity, Logistic Regression, Model evaluation.

Python Implementation (Linear Regression):
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score, mean_squared_error

# Simple Linear Regression
np.random.seed(42)
X = np.random.rand(100, 1) * 10
y = 2.5 * X.squeeze() + np.random.randn(100) * 2 + 5

# Fit model
model = LinearRegression()
model.fit(X, y)
y_pred = model.predict(X)

# Metrics
r2 = r2_score(y, y_pred)
rmse = np.sqrt(mean_squared_error(y, y_pred))

print(f"Coefficient: {model.coef_[0]:.4f}")
print(f"R-squared: {r2:.4f}")
